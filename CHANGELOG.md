### 🛠️ Технические решения

- ✅ Python 3.8+ для бэкенда
- ✅ Flask + Flask-CORS для API
- ✅ aiogram 3.x для Telegram бота
- ✅ Vanilla JavaScript (ES6+) для фронтенда
- ✅ SQLite с SQLAlchemy-подобным ORM
- ✅ CSS3 с градиентами и анимациями
- ✅ Асинхронные операции (asyncio)

### 🐛 Известные ограничения

- ⚠️ Максимум 8 операторов (по количеству цветов)
  - Решение: добавить больше эмодзи-цветов
- ⚠️ SQLite подходит для до 1000 пользователей
  - Решение: миграция на PostgreSQL
- ⚠️ Бот использует polling (опрос), а не webhooks
  - Решение: миграция на webhooks для Production
