## 🚀 СПОСОБЫ ЗАПУСКА

### 1. Локально (САМЫЙ БЫСТРЫЙ)
```bash
bash setup.sh
# Затем в 2 терминалах:
python web_app.py
python bot.py
```
Время: 5 минут
URL: http://localhost:5000
Код админа: 0000

### 2. На сервере (PRODUCTION)
Выберите один из вариантов:
- Render.com (рекомендуется) → DEPLOYMENT_RENDER.md
- Railway.app (проще всех) → DEPLOYMENT_RAILWAY.md
- Heroku (в прошлом) → используйте Procfile
- Собственный VPS → любой хостинг с Python

Время: 10-15 минут
Цена: $0-7/месяц
Доступность: 24/7

### 3. На Android (TERMUX)
```bash
# F-Droid → Termux
pkg install python pip git
git clone ...
python web_app.py & python bot.py
```
Время: 20 минут
Цена: Бесплатно
Доступность: 24/7 на телефоне
