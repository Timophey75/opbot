# 📚 ПОЛНОЕ РУКОВОДСТВО ПОЛЬЗОВАТЕЛЯ

## 🚀 БЫСТРЫЙ СТАРТ

### Для администратора:

1. **Установка:**
   ```bash
   cd opbot
   python -m venv venv
   source venv/bin/activate  # Windows: venv\Scripts\activate
   pip install -r requirements.txt
   python init.py
   ```

2. **Запуск (в двух разных терминалах):**
   - Терминал 1: `python web_app.py`
   - Терминал 2: `python bot.py`

3. **Вход:**
   - Откройте Telegram бот
   - Нажмите кнопку мини-приложения
   - Введите код: **0000** (код администратора)

### Для оператора:

1. Админ создает оператора в приложении
2. Админ дает оператору 4-значный код
3. Оператор вводит код в мини-приложении

---

## 🎮 УПРАВЛЕНИЕ РАСПИСАНИЕМ (Админ)

### Календарь

1. **Откройте "Календарь"** в главном меню
2. **Навигация:**
   - `⬅️ Назад` - предыдущий месяц
   - `📍 Сегодня` - вернуться на текущий месяц
   - `Вперед ➡️` - следующий месяц

3. **День:**
   - Нажмите на число в календаре
   - Увидите все события на этот день
   - Под каждым событием эмодзи оператора (цвет)

### Создание мероприятия

1. **Выберите день** в календаре
2. **Нажмите** "➕ Создать мероприятие утром/вечером"
3. **Выберите время:**
   - Стандартное: 10:00 (утро) или 19:00 (вечер)
   - Своё: введите в формате ЧЧ:МИ (11:30)
4. **Введите название** (например: "Озвучка программы")
5. **Нажмите "✅ Создать"**

### Назначение оператора

1. **В день события** нажмите **"➕ Назначить"** (или "↩️ Переназначить")
2. **Выберите оператора** из списка
3. Оператор получит уведомление в Telegram

### Удаление мероприятия

1. **В день события** нажмите **"🗑️ Удалить"**
2. **Подтвердите** удаление

---

## 👥 УПРАВЛЕНИЕ ОПЕРАТОРАМИ

### Добавление оператора

1. **Откройте "Операторы"** в главном меню
2. **Нажмите** "➕ Добавить оператора"
3. **Введите:**
   - Имя (например: Иван)
   - Фамилию (например: Петров)
4. **Система сгенерирует:**
   - 4-значный код доступа
   - Уникальный цвет (эмодзи)
5. **Передайте код оператору**

### Редактирование оператора

1. **Откройте "Операторы"**
2. **Нажмите** "✏️" рядом с оператором
3. **Измените имя/фамилию**
4. **Нажмите "✅ Сохранить"**

### Удаление оператора

1. **Откройте "Операторы"**
2. **Нажмите** "🗑️" рядом с оператором
3. **Подтвердите удаление**
4. Оператор больше не может войти

## 🔔 НАПОМИНАНИЯ

### Типы напоминаний

1. **⏰ За сутки**
   - Напомнит о смене за 24 часа
   - Время: вы выбираете

2. **🌅 Напомнить утром**
   - Напомнит в день смены утром
   - Стандартное время: 10:00
   - Можно изменить на любое

3. **🌆 Напомнить вечером накануне**
   - Напомнит вечером перед сменой
   - Стандартное время: 19:00
   - Можно изменить на любое

### Включение/отключение напоминаний

1. **Откройте "Напоминания"** в меню
2. **Для каждого типа:**
   - ✅ Включено/отключено (переключатель)
   - Установите время утра
   - Установите время вечера
3. **Изменения сохраняются автоматически**

## 📱 ДЛЯ ОПЕРАТОРОВ

### Просмотр расписания

1. **Откройте "Календарь"**
2. **Дни с вашими сменами** показаны с эмодзи вашего цвета
3. **Нажмите на день** для просмотра деталей
4. **Увидите:**
   - Название мероприятия
   - Время
   - Всех назначенных операторов

### Отказ от смены

1. **Откройте день** со своей сменой
2. **Укажите причину:** "Больной", "Срочные дела" и т.д.
3. **Нажмите "✅ Подтвердить отказ"**
4. **Админ получит уведомление** в Telegram
5. ⚠️ Админ должен переназначить другого оператора

### Личные напоминания

- **Закройте "Напоминания"** в меню
- Установите свои времена напоминаний
- **Не забывайте про смены!** 🔔

## 🔐 КОДЫ ДОСТУПА

### Администратор
- **Код:** 0000 (по умолчанию)
- **Может изменить** в файле `.env`
- **ВАЖНО:** Сохраняйте в безопасности!

### Операторы
- **Генерируются автоматически** (4 цифры)
- **Каждому - уникальный код**
- **Передавайте только нужному человеку**

### Цвета (эмодзи)

Каждому пользователю назначается уникальный цвет:

🟦 Синий     🟧 Оранжевый    🟥 Красный    ⬛️ Чёрный
🟩 Зелёный   🟪 Фиолетовый   🟨 Жёлтый     🟫 Коричневый

**Всего 8 цветов** - максимум 8 пользователей (вкл. админа)

## 📲 УВЕДОМЛЕНИЯ В TELEGRAM

### Типы уведомлений

1. **Назначение на смену** 📌
   - Приходит когда админ вас назначил
   - Содержит: дату, время, название

2. **Отказ от смены** ⚠️ (только для админа)
   - Приходит когда оператор отказывается
   - Содержит: кого, когда, почему

3. **Напоминания** 🔔 (по настройкам)
   - За сутки, утром, вечером накануне
   - В зависимости от ваших настроек

## 🆘 ЧАСТО ВСТРЕЧАЮЩИЕСЯ ПРОБЛЕМЫ

### "Неверный код"
- ✅ Проверьте, правильно ли введен код
- ✅ Для админа: 0000
- ✅ Для операторов: попросите у админа

### "Мероприятия не сохраняются"
- ✅ Проверьте интернет соединение
- ✅ Перезагрузите приложение
- ✅ Попробуйте еще раз

### "Не приходят уведомления"
- ✅ Убедитесь, что напоминания включены
- ✅ Проверьте время в настройках
- ✅ Позволяйте уведомления от бота

### "Не могу назначить оператора"
- ✅ Только админ может назначать
- ✅ Оператор должен существовать в системе
- ✅ Попробуйте перезагрузить приложение

- Используйте полноценную БД (PostgreSQL)
- Или создайте несколько инстансов
