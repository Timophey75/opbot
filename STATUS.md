## 🔑 ПЕРВАЯ СЕССИЯ

### Для администратора

1. Запустите приложение (см. выше)
2. Откройте http://localhost:5000
3. Введите код: **0000**
4. Добавьте первого оператора (меню "Операторы")
5. Создайте первое событие в календаре
6. Назначьте оператора на событие

✅ Готово! Приложение работает!

### Для оператора

1. Получите 4-значный код от администратора
2. Откройте приложение
3. Введите свой код
4. Посмотрите свое расписание в календаре

✅ Готово! Оператор может видеть смены и отказываться!

## 🎓 ПРИМЕРЫ ИСПОЛЬЗОВАНИЯ

### Например: Видеостудия О"ботель"

1. **Администратор:**
   - Создает расписание озвучивания на месяц
   - Добавляет операторов (Иван, Мария, Алексей, Петя и т.д.)
   - Назначает для каждого дня кто озвучивает
   - Получает уведомления если кто-то отказался

2. **Operatore (Иван):**
   - Видит в приложении когда у него смены
   - Может отказаться и указать причину
   - Получает напоминания перед сменой в Telegram

3. **Результат:**
   - Расписание всегда актуально
   - Никто не забывает о сменах
   - Админ в курсе всех отказов
   - Процесс автоматизирован
